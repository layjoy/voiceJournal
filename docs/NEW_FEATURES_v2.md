## 🎯 使用场景

### 场景1: 日常记录
录音 → 转文字 → 情绪分析 → 添加标签 → 保存

### 场景2: 回顾反思
查看情绪日历 → 发现模式 → AI深度分析 → 获取建议

### 场景3: 情绪疏导
感到焦虑 → 打开AI助手 → 对话疏导 → 获得支持

### 场景4: 数据分析
查看图表 → 发现趋势 → 关联健康数据 → 调整行为

### 场景5: 快速记录
Widget快速录音 → 转待办 → 设置提醒

### 场景6: 分享交流
匿名分享 → 获得共鸣 → 互相鼓励
