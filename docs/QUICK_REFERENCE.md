# 声音日记 - 快速参考

## 🚀 快速开始

### 1. 打开项目
```bash
# 使用Android Studio
File -> Open -> /Users/admin/Projects/voiceJournal
```

### 2. 配置SDK (可选)
```kotlin
// 讯飞语音SDK
app/libs/Msc.jar

// Claude AI API
ClaudeAIService.kt: API_KEY = "YOUR_KEY"
```

### 3. 运行
```bash
# 点击Run按钮或
./gradlew installDebug
```
