# 更新日志

## [1.0.0] - 2026-04-19

### 新增功能
- 🎙️ 声音录制功能
  - 高质量音频录制（AAC编码，44.1kHz采样率）
  - 实时音量监测
  - 录音时长显示
  
- 🌊 3D声波可视化
  - 三层波形动画
  - 根据音量动态调整振幅
  - 流畅的动画效果
  
- 📝 AI语音识别
  - 集成讯飞语音识别SDK
  - 自动转文字
  - 支持中文普通话
  
- 😊 情绪分析
  - 基于关键词的情绪识别
  - 支持6种情绪：开心、难过、平静、兴奋、焦虑、平和
  - 自动情绪配色
  
- ⏰ 时光胶囊
  - 设置解锁时间
  - 定时通知提醒
  - 锁定/解锁状态显示
  
- 📅 时间轴
  - 按时间倒序显示所有日记
  - 情绪标签
  - 删除功能

### 技术特性
- Jetpack Compose UI
- MVVM架构
- Room数据库持久化
- Kotlin Coroutines异步处理
- Material Design 3设计
- 深色主题

### 权限要求
- 录音权限
- 通知权限（Android 13+）
- 存储权限（Android 9及以下）

### 已知问题
- 模拟器上录音功能可能不稳定，建议真机测试
- 首次使用需要联网激活讯飞SDK

### 系统要求
- 最低: Android 8.0 (API 26)
- 推荐: Android 12+ (API 31+)
- 存储空间: 至少50MB
